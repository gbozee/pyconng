About PyCon Nigeria

PyCon Nigeria is the leading conference for Python developers and enthusiasts across Nigeria. Organized by the Python Nigeria community, it is a not-for-profit event run entirely by volunteers dedicated to growing the Python ecosystem in the country. Now in its 5th year, PyCon Nigeria has established itself as the largest annual gathering of the Python community in Nigeria. The conference provides an opportunity for users to learn, network and collaborate on how to use Python to solve local problems. With over 500 participants expected this year, the event continues to gain popularity among developers, data scientists, educators, engineers and tech companies. PyCon Nigeria demonstrates the power of community-led initiatives and reflects the rapid growth of Python in powering innovation across multiple industries in Nigeria. The fully packed agenda focuses on real-world use cases and local opportunities to apply Python, making this a must-attend event for anyone looking to advance their Python skills or connect with like-minded professionals from the community.
